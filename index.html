Cambio entre pantallas en Android Studio

En Android Studio, el cambio entre pantallas (o actividades) se realiza utilizando la clase Intent. Un Intent es un objeto que permite la comunicación entre diferentes componentes de una aplicación, como actividades. Para realizar el cambio de una actividad a otra, debes seguir estos pasos:

Paso 1: Crear las actividades

Si aún no tienes las actividades creadas, puedes agregar una nueva actividad en Android Studio:

  1. Haz clic derecho sobre el paquete de tu proyecto en el explorador de proyectos.
  2. Selecciona New > Activity > Empty Activity.
  3. Dale un nombre a la nueva actividad (por ejemplo, SecondActivity).

Paso 2: Configurar el Intent para el cambio de pantalla

En la actividad actual desde la que quieres cambiar de pantalla, utiliza el siguiente código para iniciar la nueva actividad con un Intent.

// Dentro de un evento, por ejemplo, al presionar un botón
Intent intent = new Intent(MainActivity.this, SecondActivity.class);
startActivity(intent);

Este código crea un Intent que especifica que quieres cambiar de MainActivity a SecondActivity y luego se llama al método startActivity() para iniciar la nueva pantalla.

Paso 3: Configurar la interfaz de usuario

Asegúrate de tener un botón u otro elemento que inicie el cambio de actividad. En el archivo activity_main.xml, agrega un botón:

<Button
    android:id="@+id/btnChangeScreen"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:text="Cambiar Pantalla" />

En el código de la actividad, configura un OnClickListener para ese botón:

Button btnChangeScreen = findViewById(R.id.btnChangeScreen);
btnChangeScreen.setOnClickListener(new View.OnClickListener() {
    @Override
    public void onClick(View v) {
        Intent intent = new Intent(MainActivity.this, SecondActivity.class);
        startActivity(intent);
    }
});

Paso 4: Declarar la segunda actividad en el archivo AndroidManifest.xml

Cuando creas una nueva actividad, Android Studio debería agregarla automáticamente al archivo AndroidManifest.xml. Asegúrate de que el archivo tenga algo como esto:

<activity android:name=".SecondActivity" />

Resumen

  1. Crea las actividades necesarias.
  2. Usa Intent para cambiar de una actividad a otra.
  3. Configura un evento (como un botón) para ejecutar el cambio.
  4. Declara todas las actividades en el AndroidManifest.xml.

¡Y eso es todo! Con estos pasos, lograrás cambiar de pantalla en tu aplicación Android.
